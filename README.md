## 🎯 Project Objective

The objective of this project is to design and develop an advanced, interactive COVID-19 analytics dashboard in Excel that integrates case data, testing metrics, and vaccination statistics into a unified analytical framework.

The project aims to:

* Monitor pandemic trends using time-series analysis
* Engineer dynamic KPIs for real-time performance tracking
* Apply predictive forecasting for future case estimation
* Enable state-level drill-down analysis
* Support data-driven public health decision-making

---

## 📊 Dataset Overview

The project integrates three primary structured datasets:

* **SummaryData** – Daily confirmed, recovered, deceased, and positive cases
* **TestingData** – State-wise daily testing sample counts
* **VaccineData** – Daily cumulative vaccination data

### Key Attributes Included:

* Date
* State / Union Territory
* Confirmed Cases
* Deaths
* Recovered Cases
* Positive Cases
* Total Samples Tested
* Total Doses Administered
* Age Group
* Population

The datasets were cleaned, validated, merged using composite keys, and structured for advanced analytical modeling.

---

## 📈 Dashboard Features

*  National Case Trend & Monthly Wave Analysis
*  Positive Rate & Testing Efficiency Tracking
*  Vaccination Rate & Recovery Rate Comparison
*  30-Day Forecast using Exponential Smoothing (ETS)
*  State-wise Contribution & Top 5 High Burden States
*  Dynamic State Selection Dropdown for Drill-Down Analysis
*  Automated Risk Classification (High / Medium / Low)
*  Weekday Seasonality Detection
*  Dynamic CFR Threshold Filtering

---

## 🔍 Key Insights

* COVID-19 followed distinct wave patterns rather than steady growth.
* Tuesday case dips indicate administrative reporting lag, not reduced spread.
* States with higher vaccination rates generally maintained stronger recovery trends.
* Forecast modeling provided upper-bound case projections for hospital planning.
* CFR-based filtering better identifies vulnerable states than raw case counts.

---

## 💼 Business Impact
---
* Enabled proactive hospital capacity planning using forecast intervals.
* Supported targeted resource allocation based on severity ratios.
* Improved visibility into state-wise outbreak intensity.
* Provided dynamic risk stratification for regulatory monitoring.
* Demonstrated how Excel can function as a full-scale analytical engine without external tools.
